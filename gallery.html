How to Add More Photos

Step 1

Go to your GitHub repository → assets/images/ → click Add file → Upload files

Step 2

Name your file clearly (e.g. lab-setup-2025.jpg) and upload it

Step 3

Open gallery.html → pencil icon to edit → copy any gal-item block → update the filename and caption

Step 4

Click Commit changes → photo appears live in ~60 seconds. No coding needed!
